# 9.9 预定收款 + 登记（最简方案）

不用开发，今天就能搭好。

## 步骤
1. **收款**：微信"收款码"或"群收款"。备注让对方写"反馈助手预定"。
2. **登记表**：用 腾讯问卷 / 金数据 / 微信表单 建一个表，字段 5 个：
   - 微信昵称/称呼
   - 教学年级/课程类型（如 一年级同步 / 数学思维 / 奥数启蒙）
   - 学员人数（用来定价分档）
   - 联系方式
   - 已付 9.9（是/否）
3. **判定（验证红线）**：
   - 2 周内拿到 **≥5 个 9.9 预定** → 立刻进入 MVP 开发（接真实大模型）。
   - 只有 **0–1 个** → 别继续写代码，回来换品类或换方向。

## 给所有付费老师的承诺
- 开发期间继续免费手工帮你生成反馈。
- 小工具上线后永久半价。
- 不满意全额退。

让老师零风险，你拿到真实付费验证。

## 本周目标
- 私信 30 个老师
- 免费服务 10 个
- 拿到 5 个 9.9 预定
