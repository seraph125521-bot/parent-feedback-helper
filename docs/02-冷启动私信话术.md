# 找机构 + 私信冷启动话术

## 去哪找（按效率排序）
1. 抖音/小红书同城搜：`少儿数学 + 城市`、`小学数学辅导`、`数学思维课`、`奥数启蒙`，刷机构号和老师个人号，主页常留微信/电话。
2. 小红书搜：`数学老师 家长反馈`、`课后反馈 怎么写`、`数学课家长沟通`，会刷到正在为这事头疼的老师，精准。
3. 本地教培老板群 / 同城机构群（找一个加进去潜水）。

## 私信话术（关键：先送价值，不像推销）

**第一句（破冰，不提产品）：**
> 老师您好～看到您是教少儿数学/小学数学的。我在做一个帮老师自动写家长课后反馈的小工具，正找几位老师免费试用。您平时课后会给家长发反馈吗？发起来累不累？

**如果回复"会发/挺累的"：**
> 那太对口了。这样，您把今天讲的知识点、班级整体情况和每个孩子的一句表现发我（语音也行），我免费帮您生成一周的家长反馈，您直接复制发群里就行，看看好不好用、家长反应怎么样。完全免费，就想听听真实老师的意见。

**交付几次、对方觉得好用后（转化）：**
> 这个我准备做成手机上能自己用的小工具，老师课后填几个词就自动出一整个班的反馈，49 元/月。现在招几个共创老师，9.9 元预定一个月、永久半价。您要不要占个名额？不满意随时退。

## 话术原则
- 全程"帮你解决问题"姿态，不催不逼。
- 先白送 3 次建立信任，再谈钱。
- 用 9.9 预定来筛真需求。
- 现在有了能用的 H5 小工具，演示时直接发链接让老师自己点一下，转化率更高。
